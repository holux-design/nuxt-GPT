# Getting Started

`nuxt-GPT` is a batteries-included module for easy access to the power of chatGPT from your codebase.


## Auto install


```bash [npx]
npx nuxi module add nuxt-gpt
```


## Install manually

1. Install from npm

::code-group

  ```bash [npm]
  npm i nuxt-gpt
  ```
::

2. Add module to your `nuxt.config.ts`

::code-group

  ```ts [nuxt.config.ts]
  modules: [
    'nuxt-gpt'
  ]
  ```
::
